# Scope

## Include

- 문서 업로드 intake
  - 등기부등본 PDF
  - 임대차 계약서 PDF 또는 이미지
  - 문서 종류 선택 또는 자동 분류
- 문서 처리 파이프라인
  - 원본 저장
  - 텍스트 추출
  - 필드 추출
  - 신뢰도 계산
  - 사용자 검토/승인
  - 승인된 필드의 기존 진단 입력값 반영
- 등기부등본 추출 후보
  - 부동산 주소
  - 발급일 또는 열람일
  - 소유자명
  - 갑구 위험 신호: 압류, 가압류, 가처분, 경매개시결정
  - 을구 위험 신호: 근저당권, 전세권, 임차권등기, 신탁 관련 문구
  - 채권최고액 또는 선순위채권 금액 후보
- 임대차 계약서 추출 후보
  - 임대인명
  - 임차인명은 저장 대상에서 기본 제외하거나 마스킹
  - 소재지/호수
  - 계약 유형
  - 보증금
  - 월세
  - 계약일
  - 입주 예정일 또는 임대차 시작일
  - 임대차 종료일
  - 특약 핵심 문장 후보
- 불일치 검증
  - 계약서 주소와 등기부등본 주소 비교
  - 계약서 임대인과 등기부등본 소유자 비교
  - 계약서 보증금과 진단 입력값 비교
  - 발급일이 오래된 등기부등본 경고
- 운영 품질
  - 문서 처리 상태와 retry
  - 추출 실패 사유
  - 감사 로그
  - 원본 삭제 정책을 위한 상태/메타데이터

## Exclude For First Production Slice

- 서버가 사용자를 대신해 인터넷등기소 로그인, 인증, 결제, 공동인증서 처리를 수행하는 기능
- 주민등록번호 원문 저장
- 법률 자문 또는 계약 가능/불가능 확정 판정
- 자동 추출 결과를 사용자 확인 없이 곧바로 최종 분석값으로 확정하는 흐름
- 외부 OCR 공급자 하나에 강하게 결합된 도메인 로직

## Extension Points

- OCR provider adapter
- 공식 등기 확인 또는 검증 provider adapter
- 계약서 특약 위험 문장 classifier
- 문서 retention scheduler
- PII redaction service

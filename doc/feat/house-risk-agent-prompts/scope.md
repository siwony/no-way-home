# Scope

## Include

- 계약 유형: `JEONSE`, `MONTHLY_RENT`, `SEMI_JEONSE`
- 주택 유형: `APT`, `OFFICETEL`, `VILLA`, `MULTI_FAMILY`, `MULTI_HOUSEHOLD`, `UNKNOWN`
- 필수 입력: 주소, 계약 유형, 보증금, 월세, 계약 예정일, 입주 예정일, 임대인 이름, 주택 유형
- 선택 입력: 중개 여부, 공인중개사 정보, 예상 매매가, 예상 전세가, 등기부등본 PDF, 건축물대장 PDF
- 외부/업로드 자료: 등기부등본, 건축물대장, 시세/실거래가, 계약 정보
- 위험 분석: 등기 권리, 건축물 용도/위반 여부, 전세가율, 총 위험 노출 비율, 보수적 경매 회수 시뮬레이션
- 결과 제공: 리포트, 위험 문구, 추가 확인 항목, 계약 체크리스트

## Exclude

- 임대인 국세/지방세 체납 여부 조회
- 법률 자문
- 보증보험 가입 가능성의 확정 판정
- 주민등록번호 수집 또는 저장
- 인증서, 공동인증서, 간편인증 토큰 저장
- “계약해도 안전하다” 같은 확정적 안전 판정

## Out Of Scope Handling

- 세금 체납 조회가 필요한 경우 사용자가 별도 확인해야 한다고 안내한다.
- 법률 판단이 필요한 경우 전문가 상담이 필요하다고 안내한다.
- 주택 유형이 `UNKNOWN`이면 리포트에 “주택 유형 확인 필요”를 표시한다.

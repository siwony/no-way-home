# Goal

사용자가 전세, 월세, 반전세 계약 전 특정 주택의 계약 위험 신호를 확인할 수 있도록 주택 계약 위험도 진단 서비스를 구현한다.

이 기능은 등기부등본, 건축물대장, 시세/실거래가, 계약 정보를 기반으로 위험도 리포트를 생성한다. 결과는 확정적 안전 판정이 아니라 현재 확인된 자료 기준의 위험 신호 탐지 결과여야 한다.

## User Problem

- 계약 전 등기부등본, 건축물대장, 시세 정보를 종합해서 판단하기 어렵다.
- 근저당, 압류, 가압류, 위반건축물, 전세가율 같은 위험 요소를 사용자가 직접 해석하기 어렵다.
- 계약 전/직전/후에 무엇을 추가로 확인해야 하는지 놓치기 쉽다.

## Core Value

- 위험도 등급을 `SAFE`, `CAUTION`, `DANGER`, `CRITICAL`로 분류한다.
- 핵심 위험 사유와 추가 확인 항목을 사용자가 이해할 수 있는 문장으로 제공한다.
- 보증금, 선순위채권, 추정 시세를 이용해 보증금 회수 위험을 계산한다.
- 계약 단계별 체크리스트를 제공한다.

## Final Output

- 위험도 등급
- 위험 사유 목록
- 등기부등본 분석 결과
- 건축물대장 분석 결과
- 보증금 위험 분석
- 보수적 회수 시뮬레이션
- 계약 전/직전/후 체크리스트
